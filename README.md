# PMO Dashboard - نظام إدارة المشاريع

نظام إدارة المشاريع والموظفين لمركز التأمين الصحي الوطني

## المميزات
- إدارة الموظفين والمشاريع
- المهام الأسبوعية
- متابعة الحضور والإجازات
- المبادرات والإنجازات
- تقارير الطباعة

## التقنيات
- Flask
- SQLAlchemy
- SQLite
- Tailwind CSS

## التشغيل محلياً
```bash
pip install -r requirements.txt
python app.py
```
