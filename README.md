## 📋 المميزات

### 1. لوحة التحكم الرئيسية
- 5 بطاقات KPI (موظفين، مشاريع، محافظ، مبادرات، نسبة إنجاز)
- 4 رسوم بيانية تفاعلية (Bar, Pie, Horizontal Bar, Doughnut)
- شعار مركز التأمين الصحي الوطني
- تاريخ اليوم بالعربي
- جدول موظفين مع شريط تقدم لنسبة الإنجاز

### 2. إدارة الموظفين (5 موظفين)
| الموظف | المحافظ |
|--------|---------|
| شكرية كنكار | المالية، الحوكمة، المنافع |
| نايف الجلال | الشراء الاستراتيجي، الاستدامة المالية، القانونية |
| عبد العزيز المعجل | المراجعة الداخلية، تقييم التقنيات |
| صالح الغفيلي | الصحة الحكيمة، الترميز الطبي، التقنية |
| مشعل محمد | الموارد البشرية، التواصل |

### 3. المهام الأسبوعية (13 مهمة)
1. تحديث الأنظمة والمنصات المتعلقة بالمشاريع
2. تحديث الخطة التفصيلية للمشروع
3. حل التحديات والمخاطر
4. متابعة وإغلاق أوامر التغيير
5. إرسال تقرير أسبوعي لمالك المحفظة
6. التنبيه بثلاثة أسابيع بحلول قرب تسليم المخرجات
7. متابعة تسليم المخرجات
8. تحديث الخطة المالية الرئيسية
9. تصدير شهادات الإنجاز للمالية
10. متابعة وتحديث المشاريع التشغيلية
11. رفع جميع الوثائق في نظام ميسر
12. المبادرات والإنجازات
13. الالتزام بمصفوفة الصلاحيات التشغيلية

### 4. الحضور والغياب
- 4 بطاقات إحصائيات (أيام العمل، نسبة الحضور، إجازات سنوية، إجازات مرضية)
- جدول رصيد إجازات مع شريط تقدم
- دليل رموز ملون
- تقويم شهري تفاعلي
- أيقونات: ✅ حاضر، 🌴 سنوية، 🤒 مرضية، 🏠 عن بعد، ⏱️ أخرى

### 5. المبادرات والإنجازات
- اسم المبادرة، الوصف، تاريخ البداية والنهاية، الحالة

### 6. المهام الإضافية
- عنوان المهمة، الوصف، التواريخ، الحالة

### 7. الأهداف السنوية
- محمية بكلمة مرور إضافية

### 8. تقارير الطباعة
- اختيار نوع التقرير (كامل/مشاريع/مبادرات/مهام)
- اختيار الموظف
- شعار في الأعلى
- جداول ملونة
- CSS للطباعة

### 9. تقييم الأداء
- 5 مستويات (غير مرضي → متميز)

## 🚀 التشغيل المحلي

### المتطلبات
- Python 3.8+
- pip

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/jwaheralameer77-oss/pmo-dashboard-SA.git
cd pmo-dashboard-SA

# تثبيت المتطلبات
pip install -r requirements.txt

# تشغيل التطبيق
python app.py
```

### الوصول
افتح المتصفح على: `http://127.0.0.1:5000`

### بيانات الدخول
- اسم المستخدم: `admin`
- كلمة المرور: `pmo2024`
